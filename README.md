# LeafLogic App

##Project Description

LeafLogic is a Flutter mobile application that helps users choose suitable plants based on their environment and preferences.
The app allows users to browse plants from a Supabase database, take a quiz to get recommendations, and add plants to a shopping cart.

---

##Featurs

- User authentication (Sign up & Login) using Supabase
- Retrieve plant data from Supabase database
- Plant recommendation quiz
- Add and remove plants from cart
- Responsive and organized UI
- Navigation between screens
